#### Objetivo:
```
?>cat z

?>
```

------------------------------------------
#### Criando o arquivo
Há várias formas de se criar um arquivo de texto, seja pelo vim, cat, VSCode etc.

Por exemplo:
```
cat > z
Z

ctrl + D para salvar
```

> Importante não se esquecer da quebra de linha.


------------------------------------------
Para mais detalhes e melhor entendimento, consulte:
```
$ man cat
```
